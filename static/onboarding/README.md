# Onboarding screenshots

Drop these PNGs in this folder to replace the CSS-mocked placeholders on the
dashboard tour:

- `step-1.png` — Create-league dialog on the dashboard.
- `step-2.png` — Madden Companion App **Settings → Export** with the export URL pasted.
- `step-3.png` — Dashboard showing standings/rosters populated after sim.
- `step-4.png` — Discord channel with the bot posting standings or weekly headline.

Recommended dimensions: **1280x800** (or any 16:10 aspect). Keep individual files
under ~250 KB. PNG or JPG both work; the templates reference `.png` directly.

If a file is missing, the dashboard automatically falls back to a colored CSS
mockup so the layout never breaks.
